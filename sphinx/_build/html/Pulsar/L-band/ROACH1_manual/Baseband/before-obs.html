Before observing¶

Notes : The following instructions are for observing at L-band with the ROACH1 in baseband mode. The observed bandwidth is limited to 128 MHz.

After the standard checks:

1. On a desktop open the leap0 VNC session using VNCViewer or TigerVNC and selecting leap0:1 (ask for password).

If it is not active:

  1. Log on to the LEAP cluster :

    $ ssh -X user@leap0 ask for password

  2. Launch a VNC session once logged on to leap0 :

    leap0: $ vncserver &

On the LEAP cluster (using VNC):¶

Initial setup for the LEAP cluster:

  1. Open a window (W1) and go to :

    $ ~/externalClient/Control/

    This window will be used to start and stop data acquisition.

  2. In W1: type a command to select L-band (and not P-band):

    $ ~/roach/scripts/setLband.sh
